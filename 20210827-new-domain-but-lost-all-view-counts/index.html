Posted on 2021-08-27 Edited on 2023-06-09 In 3. Life , 3.3 Blog Views:

cv-learn 도메인 이전

옛날의 cv-learn 블로그는 Notion 블로그 기반으로 운영했었다. 아마 이 당시의 블로그가 가장 cv-learn이 잘 알려지게 된 계기였지 않을까 싶다.

그러다가 여러가지 이유로 새롭게 GitHub 블로그를 시작하게 되었다.

물론 이전 블로그에서 새 블로그로 글을 옮기려고 했지만, 노션->Hexo로의 이동은 생각보다 큰 작업이였고, 짧은 시간동안 할 수 있는 작업이 아니였다.

그래서 글을 옮기고 또 새 글을 적는동안만큼은 노션 블로그를 유지하기로 했다. 일일 방문자의 양을 고려했을 때 그 편이 더 좋을 것 같았기 때문이다.

그리고 시간이 지나고, GitHub 블로그도 꽤 이제 블로그다운 구실을 갖추게 되었다.

도메인 이전

CloudFlare 서비스를 통해 도메인을 옮겼다.

생각처럼 쉬운 과정은 아니였지만, 그래도 1~2시간 만에 된거면 만족한다.

하지만 알아챈 것중에 정말 아쉬운 점은…

모든 글들에 대한 조회수 카운트, 블로그 전체 고유 방문자 수, 블로그 전체 사이트 조회 수가 리셋이 된거다.

아마 url에 고정되서 static website 캐싱 작업이 되었던 것 같은데… 새 도메인으로 옮기면 이 데이터가 이전이 안되는 것 같다.

그래서 새출발이 되어버렸다 아뿔싸~~~
