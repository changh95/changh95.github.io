SHA256

  • Hash
    • вҖңA unique fixed length string, meant to identify a piece of data. They are created by placing said data into a вҖҳhash functionвҖҷвҖқ
    • мһ…л Ҙ лҚ°мқҙн„°м—җ лҢҖн•ҙ мң мқјн•ң hashк°Җ мғқм„ұлҗңлӢӨ.

Block

  • Blockм—җлҠ” block number, nonce, dataк°Җ мһҲлӢӨ.
    • мқҙм ң HashлҠ” block number, nonce, dataлҘј лӘЁл‘җ н•Ёк»ҳ м„һм–ҙм„ң л§Ңл“Өм–ҙлӮёлӢӨ.
    • Block numberлҠ” blockchainм—җ мҳ¬лқјк°Ҳ л•Ң blockмқҳ мҲңм„ңмқҙлӢӨ
    • NonceлҠ” Hashмқҳ мІ« м•һ 4мһҗлҰ¬лҘј 0000мңјлЎң л§Ңл“ңлҠ” мҲ«мһҗ к°’мқҙлӢӨ.
      • Dataк°Җ л“Өм–ҙмҷ”мқ„ л•Ң hashмқҳ м•һ 4мһҗлҰ¬к°Җ 0000мқҙ лҗ  нҷ•лҘ мқҖ көүмһҘнһҲ лӮ®лӢӨ.
      • Nonceк°Җ лӘҮмқҙ лҗ м§Җ мҡ°лҰ¬лҠ” лӘЁлҘёлӢӨ. 0л¶Җн„° мӢңмһ‘н•ҙм„ң 1м”© лҚ”н•ҙк°Җл©°, hashмқҳ мІ« м•һ 4мһҗлҰ¬к°Җ 0000мқҙ лҗ  л•Ң к№Ңм§Җ кі„мӮ°н•ңлӢӨ.
        • мқҙ кіјм •мқ„ мұ„көҙ - Miningмқҙлқјкі  н•ңлӢӨ.
        • мұ„көҙ мһ‘м—…мқҳ кі„мӮ°лҹүмқҖ л§ҺлӢӨ.
    • мқҙл ҮкІҢ мұ„көҙмқ„ н•ЁмңјлЎңмҚЁ nonceлҘј м°ҫлҠ” мһ‘м—…мқ„ Proof of work (PoW)лқјкі  н•ҳкё°лҸ„ н•ңлӢӨ.

hashмқҳ мІ« м•һмһҗлҰ¬к°Җ 0000мқҙ лҗңлӢӨлҠ” кІғмқҖ м—¬кё°м„ңмқҳ мҳҲм ңмқҙлӢӨ.

Blockchain

  • BlockchainмқҖ blockмқҙ м—°кІ°лҗҳм–ҙмһҲлӢӨ.
    • к°Ғк°Ғмқҳ BlockмқҖ мқҙм ң previous blockмқҳ hashлҸ„ н•Ёк»ҳ лҚ°мқҙн„°лЎң к°Җм§„лӢӨ.
    • Blockchainмқҳ мІ« blockмқҖ Genesis blockмқҙлқјкі  н•ҳл©°, мқҙ blockмқҖ prev hashлҘј к°Җм§Җм§Җ м•ҠлҠ”лӢӨ.
    • мӨ‘к°„ blockмқҳ лҚ°мқҙн„°лҘј мҲҳм •н•ҳл©ҙ, н•ҙлӢ№ blockм—җм„ңл¶Җн„° chainмқҳ лҒқ blockк№Ңм§Җ miningмқ„ н•ҙм•јн•ңлӢӨ.
      • мҷңлғҗн•ҳл©ҙвҖҰ мҡ°м„  н•ҙлӢ№ blockмқҳ hashк°Җ 0000мңјлЎң мӢңмһ‘н•ҳм§Җ м•Ҡмқ„ кІғмқҙкі , мқҙлҘј мң„н•ҙ мғҲлЎӯкІҢ nonceлҘј мұ„көҙн•ҙм•јн•ңлӢӨ.
      • hashк°Җ 0000мқҙ лҗҳлҠ” nonceлҘј мұ„көҙн•ҳл©ҙ, к·ё лӢӨмқҢ blockмқҳ prev hash лҚ°мқҙн„°к°Җ л°”лҖҢм–ҙ hash лҚ°мқҙн„°к°Җ 0000мңјлЎң мӢңмһ‘н•ҳм§Җ м•ҠкІҢ лҗңлӢӨ.
      • мқҙ blockмқҳ nonceлҸ„ мұ„көҙн•ҳкі вҖҰ лҳҗ лӢӨмқҢ blockмқҳ prev hashк°Җ л°”лҖҢм–ҙм„ң лҳҗ nonceлҘј мұ„көҙн•ҳкі вҖҰ
    • мұ„көҙ кі„мӮ°мқҖ кі„мӮ°лҹүмқҙ көүмһҘнһҲ л§ҺлӢӨ.
      • мӨ‘к°„м—җ мһҲлҠ” block к°’мқ„ л°”кҫёл©ҙ лӢӨмҲҳмқҳ blockмқ„ мғҲлЎң мұ„көҙн•ҙм•јн•ҳкё° л•Ңл¬ём—җ кі„мӮ°лҹүмқҙ көүмһҘнһҲ л§Һм•„м§Ҳ кІғмқҙлӢӨ.

Distributed blockchain

  • лӢӨмҲҳмқҳ мӮ¬лһҢмқҙ лҸҷмқјн•ң blockchain м •ліҙлҘј к°Җм§Җкі  мһҲлӢӨкі  н•ҙліҙмһҗ.
    • мқҙ л•Ң, н•ңлӘ…мқҙ мӨ‘к°„ blockмқҳ лҚ°мқҙн„°лҘј л°”кҝ” chainмқҳ лҒқк№Ңм§Җ мғҲлЎӯкІҢ мұ„көҙн–ҲлӢӨкі  н•ҙліҙмһҗ.
    • к·ёлһҳлҸ„ лӢӨлҘё мӮ¬лһҢл“Өмқҳ blockchainмқҖ л°”лҖҢм§Җ м•Ҡм•ҳлӢӨ.
      • м—¬лҹ¬ blockchainл“Өмқ„ л№„көҗн–Ҳмқ„ л•Ң лӢӨмҲҳкІ°мқҳ blockchainмқҙ лҚ” мӢ л№ҷм„ұ мһҲлӢӨкі  нҢҗлӢЁн•  мҲҳ мһҲлӢӨ.
      • мқҙлҹ¬н•ң кө¬мЎ°лҘј нҶөн•ҙ мһ„мқҳмқҳ мң м Җк°Җ blockchain лӮҙмқҳ м •ліҙлҘј н•ҙнӮ№н•ҳкё° м–ҙл өкІҢ л§Ңл“ лӢӨ.
        • н•ҙнӮ№мқ„ н•ҳкё° мң„н•ҙм„ңлҠ” blockchainмқ„ к°Җм§„ лӘЁл“  мӮ¬лһҢмқҳ мөңмҶҢ 50%лҠ” н•ҙнӮ№мқ„ лӢӨ н•ҙм•јн•ҳкё° л•Ңл¬ёмқҙлӢӨ.
        • мў…мў…, нҠ№м • nodeм—җм„ң block м •ліҙлҘј н•ҙнӮ№н•ЁмңјлЎңмҚЁ лӢӨлҘё м •ліҙлҘј к°Җм§Җкі  мһҲмқ„ л•ҢлҠ” лӢӨмҲҳмқҳ consensusлҘј мң м§Җн•ҳкё° мң„н•ҙ н•ҙлӢ№ nodeлҘј м ңкұ°н•ҙлІ„лҰ¬кё°лҸ„ н•ңлӢӨ.

Tokens

  • Blockchainм—җ кұ°лһҳ лӮҙм—ӯм—җ лҢҖн•ң лҚ°мқҙн„°лҘј лӢҙлҠ”лӢӨкі  н•ҙліҙмһҗ.
    • кұ°лһҳ лӮҙм—ӯм—җ лҢҖн•ң лҚ°мқҙн„°лҸ„ hashлЎң ліҖнҷҳлҗ  мҲҳ мһҲлӢӨ.
    • кұ°лһҳ лӮҙм—ӯмқ„ мЎ°мһ‘н•ҳл Өкі  н•ҳл©ҙ н•ҙлӢ№ blockмқҳ hashк°Җ ліҖкІҪлҗҳл©°, мқҙнӣ„ лӘЁл“  blockл“Өмқ„ мғҲлЎӯкІҢ мұ„көҙн•ҙм•јн•ңлӢӨ.
  • ліҙлӮҙлҠ” мӮ¬лһҢкіј л°ӣлҠ” мӮ¬лһҢмқҳ м •ліҙлҠ” public keyлҘј нҶөн•ҙ л§Ңл“Өм–ҙм§„лӢӨ.
    • мқҙ public keyлҠ” private key м •ліҙм—җ Elliptic Curve Digital Signature Algorithm (ECDSA) м•Ңкі лҰ¬мҰҳмқ„ нҶөн•ҙ мғқм„ұлҗңлӢӨ.
      • Private keyлҠ” seed phraseмҷҖ account addressлҘј м„һм–ҙм„ң л§Ңл“Өм–ҙм§„лӢӨ.
    • лӢҙкі  мӢ¶мқҖ м •ліҙмҷҖ private keyлҘј н•Ёк»ҳ лҚ°мқҙн„°м—җ л„Јм–ҙм„ң ECDSA м•Ңкі лҰ¬мҰҳм—җ л„Јмңјл©ҙ message signatureк°Җ лӮҳмҳЁлӢӨ.
      • нғҖмқёмқҖ public keyк°Җ мһҲмңјл©ҙ н•ҙлӢ№ message signatureк°Җ м§„м§ңмқём§Җ м•„лӢҢм§Җ кө¬л¶„н•  мҲҳ мһҲлӢӨ.
        • мҰү, лҲ„кө¬л“ м§Җ мқҙ кұ°лһҳк°Җ м§„м§ңмқём§Җ м•„лӢҢм§Җ нҷ•мқён•  мҲҳ мһҲлӢӨлҠ” кІғмқҙлӢӨ.
      • нғҖмқёмқҖ н•ҙлӢ№ message signatureлЎңл¶Җн„° private keyлҘј м¶”лЎ н•  мҲҳ м—ҶлӢӨ.

Consensus

Consensusм—җ лҢҖн•ң мӢ¬лҸ„ к№ҠмқҖ л¬ём„ң

  • нғҲмӨ‘м•ҷнҷ” лҗң мӢңмҠӨн…ңм—җм„ң лӘЁл“  nodeл“Өмқҙ лҸҷмқҳн•  мҲҳ мһҲлҠ” лӢЁ н•ҳлӮҳмқҳ blockchainмқҳ stateлҘј м •н•ҳлҠ” мһ‘м—…мқ„ consensusлқјкі  н•ңлӢӨ.
    • л„ӨнҠёмӣҢнҒ¬мқҳ 51%к°Җ лҸҷмқҳн•ҳлҠ” blockchainмқҙ м§„м§ңлқјкі  лҜҝлҠ” л°©мӢқмқҙлӢӨ.
    • л‘җк°Җм§Җ мӨ‘мҡ”н•ң consensus л©”м»ӨлӢҲмҰҳмқҙ мһҲлӢӨ : Chain selection, Sybil resistance.
      • Chain selection
        • Nakamoto consensus
          • м§„м§ң лё”лЎқмІҙмқёкіј к°Җм§ң лё”лЎқмІҙмқёмқ„ к°Җл ӨлӮҙлҠ” л°©мӢқ
          • к°ҖмһҘ кёҙ мІҙмқёмқ„ м“°лҠ” л°©мӢқмқ„ мӮ¬мҡ©н•Ё
            • мІҙмқёмқҙ кёё мҲҳлЎқ proof of workлҘј н•ҳлҠ”лҚ°м—җ мӢңк°„мқҙ лҚ” мҳӨлһҳ кұёлҰ¬кё° л•Ңл¬ё (i.e. лҚ°мқҙн„°лҘј мүҪкІҢ л°”кҫём§Җ лӘ»н•Ё)
          • л№„нҠёмҪ”мқёкіј мқҙлҚ”лҰ¬мӣҖмқҙ мӮ¬мҡ©н•ҳлҠ” л°©мӢқ.
      • Sybil resistance
        • м—¬кё°м—җ л‘җк°Җм§Җ л°©лІ•мқҙ мһҲлӢӨ - proof of work, proof of stake
        • Proof of work
          • POW: лӘЁл“  nodeк°Җ мұ„көҙмқ„ лӘ©м ҒмңјлЎң кІҪмҹҒмқ„ н•ҳл©°, к°ҖмһҘ л№ лҘҙкІҢ powлҘј лҒқлӮҙлҠ” мӘҪмқҙ лҸҲмқ„ л°ӣлҠ” л°©мӢқ.
          • 51%мқҳ л„ӨнҠёмӣҢнҒ¬лҘј мһҘм•…н•ҳкё° мң„н•ҙм„ңлҠ”, мқҙлЎ м ҒмңјлЎң 51%мқҳ л„ӨнҠёмӣҢнҒ¬ліҙлӢӨ лҚ” л№ лҘҙкІҢ кі„мӮ°н•  мҲҳ мһҲм–ҙм•јн•ңлӢӨ.
            • лІ„лҠ” лҸҲ ліҙлӢӨ мһҘл№„ мӮ¬лҠ” лҸҲмқҙ лҚ” л§Һмқ„ мҲҳлҸ„вҖҰ
          • л№„нҠёмҪ”мқёкіј мқҙлҚ”лҰ¬мӣҖмқҙ мӮ¬мҡ©н•ҳлҠ” л°©мӢқ.
        • Proof of stake
          • Proof of workлҠ” м—җл„Ҳм§ҖлҘј л§Һмқҙ мҶҢл№„н•ҳкё° л•Ңл¬ём—җ нҷҳкІҪм—җ мўӢм§Җ м•ҠлӢӨ.
            • Proof of stakeк°Җ proof of workм—җ л“Өм–ҙк°ҖлҠ” м—җл„Ҳм§Җмқҳ 1%л§Ң мҶҢл№„н•  кІғмңјлЎң мҳҲмғҒлҗңлӢӨ.
          • Proof of stakeлҠ” nodeл“Өмқҙ к°Ғмһҗ вҖҳлӮҳлҠ” лӮҳмҒҳкІҢ көҙм§Җ м•ҠкІ мҠөлӢҲлӢӨвҖҷлқјлҠ” мқҳлҜёлЎң мҪ”мқёмқ„ мҳ¬л ӨлҶ“лҠ”лӢӨкі  н•ңлӢӨ.
            • лӮҳмҒҳкІҢ көҙл©ҙ мқҙ мҪ”мқёмқ„ лӘ°мҲҳн•ңлӢӨ.
            • м°©н•ң nodeл“Өм—җкІҢ лһңлҚӨн•ҳкІҢ мұ„көҙ мһ‘м—…мқҙ л¶Җм—¬лҗңлӢӨ.
          • 51%мқҳ л„ӨнҠёмӣҢнҒ¬лҘј мһҘм•…н•ҳкё° мң„н•ҙм„ңлҠ”, мқҙлЎ м ҒмңјлЎң л„ӨнҠёмӣҢнҒ¬м—җ мҳ¬л ӨлҶ“мқҖ мҪ”мқёл“Өмқҳ 51%ліҙлӢӨ л§Һмқҙ к°Җм§Җкі  мһҲм–ҙм•јн•ңлӢӨлҠ” кІғмқҙлӢӨ.
            • мқҙ л¶Җл¶„м—җ лҢҖн•ҙм„ң мӨ‘м•ҷнҷ”/нғҲмӨ‘м•ҷнҷ”м—җ лҢҖн•ң нҶ лЎ мқҙ мҳӨк°Җкі мһҲлӢӨ.
          • Avalanche, Solana, Polygon, Polkadot л“ұмқҙ мӮ¬мҡ©н•ҳлҠ” л°©мӢқ
          • мқҙлҚ”лҰ¬мӣҖлҸ„ ETH 2.0лЎң м—…к·ёл Ҳмқҙл“ң н•  л•Ң мқҙ л°©мӢқмқ„ мӮ¬мҡ©н•  кІғмқҙлқјкі  н•ңлӢӨ.

Proof of work & Block award

  • Proof of workлҠ” көүмһҘнһҲ л§ҺмқҖ м—җл„Ҳм§ҖлҘј мҶҢл№„н•ңлӢӨ (i.e. м—°мӮ°лҹүмқҙ л§ҺлӢӨ)
    • нҠ№м • blockм—җ лҢҖн•ң powлҘј лЁјм Җ лҒқлӮҙлҠ” nodeк°Җ лҸҲмқ„ л°ӣкІҢ лҗңлӢӨ.
      • мқҙ л•Ң, л‘җк°Җм§Җ л°©лІ•мңјлЎң лҸҲмқ„ л°ӣлҠ”лӢӨ : Transaction fee, Block award.
        • Transaction feeлҠ” Gasл№„мҷҖ лҸҷмқјн•ҳлӢӨ.
        • Block awardлҠ” лё”лЎқмІҙмқё мһҗмІҙм—җм„ң мЈјлҠ” лҸҲмңјлЎңмҚЁ, Powм—җ лҢҖн•ң ліҙмғҒмқ„ мЈјлҠ” кІғмқҙлӢӨ.
          • л№„нҠёмҪ”мқё лё”лЎқмІҙмқёмқҖ block awardлЎң л№„нҠёмҪ”мқёмқ„, мқҙлҚ”лҰ¬мӣҖ лё”лЎқмІҙмқёмқҖ block awardлЎң мқҙлҚ”лҰ¬мӣҖмқ„ мӨҖлӢӨ.
          • лё”лЎқмІҙмқёмқҙ м ңкіөн•  мҲҳ мһҲлҠ” block awardлҠ” н•ңм •лҗҳм–ҙмһҲкё° л•Ңл¬ём—җ, мӢңк°„мқҙ м§ҖлӮ мҲҳлЎқ block awardк°Җ мӨ„м–ҙл“ лӢӨ.
            • л№„нҠёмҪ”мқёмқҳ л°ҳк°җкё°лҠ” 4л…„ м •лҸ„мқҙлӢӨ.
          • Block awardлҘј нҶөн•ҙ мӢңк°„мқҙ м§ҖлӮ мҲҳлЎқ лҚ” л§ҺмқҖ лҸҲмқҙ мӢңмһҘм—җ н’ҖлҰҙ мҲҳ мһҲлҸ„лЎқ н•ҳлҠ” мӢңмҠӨн…ңмқҙлӢӨ.

н•ҙнӮ№

  • Sybil attack
    • мң м Җк°Җ лӢӨмҲҳмқҳ кі„м •мқ„ мғқм„ұн•ҙм„ң лё”лЎқмІҙмқёмқҳ consensusлҘј нҶөм ңн•ҳл ӨлҠ” мӢңлҸ„
    • н•ҙлӢ№ мң м ҖлҠ” лӘЁл“  кі„м •м—җм„ң к°Ғк°Ғ powлҘј н•ҙм•јн•ңлӢӨ. (м—„мІӯлӮҳкІҢ л§ҺмқҖ кі„мӮ°лҹү!)
