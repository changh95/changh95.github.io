std::vector<>Ήζα Ήζ┤Έθ░ ΊΛ╣ΉΕ▒Ήζ┤ Ήβε ΉΕ▒ΈΛξΊΨξΉΔΒΉζΕ ΉΜεΊΓνΈΔΡ, Έξ╝ ΉΕνΈςΖΊΧαΈινΈσ┤ Ήζ┤ Ά╕ΑΉζ┤ Ά╜ν Ά╕╕ΉΨ┤ΉπΙΆ▓ΒΈΜΙΈΜν. ΊΦΕΈκεΉΕ╕ΉΕεΆ░Α ΉΨ┤ΈΨ╗Ά▓Ν ΈσΦΈςρΈουΈξ╝ Ή▓αΈουΊΧαΈΛΦΉπΑ ΉΕνΈςΖΉζΕ ΊΧ┤ΉΧ╝ΊΧιΊΖΝΈΜΙΆ╣ΝΉγΦ.

ΉλΑ ΈΞΦ ΉηΡΉΕ╕ΊηΙ ΉΧΝΆ│ιΉΜ╢Ήε╝ΉΜι Έ╢ΕΈΥνΉζΑ, έΑεC++ cache aware programmingέΑζΉζ┤Έζ╝Ά│ι Ά╡υΆ╕ΑΈπΒ ΊΧ┤Έ│┤ΉΜεΆ▒░ΈΓα, ΉΧΕΈΜΙΈσ┤ ΉΧΕΈηα ΉηΡΈμΝΈξ╝ Έ│┤ΉΜεΈΛΦΆ▒╕ Ή╢ΦΉ▓εΈΥεΈο╜ΈΜΙΈΜν.

Έ╣ΕΉΨ┤ΉηΙΆ▒░ΈΓα, Ά╜Κ Ή░ρΉηΙΈΛΦ std::vector<>ΉΩΡ ΉΔΙΈκεΉγ┤ elementΈξ╝ ΈΕμΉζΕ ΈΧΝ, Ήγ░ΈουΈΛΦέΑο

  • Ήζ┤ΉιΕ ΈσΦΈςρΈου Έ╕ΦΈκζΉΩΡ ΉιΑΉηξΈΡαΉΨ┤ΉηΙΈΛΦ ΈςρΈΥι element ΉιΧΈ│┤Έξ╝ ΉΔΙΈκεΉγ┤ Έ╕ΦΈκζΉε╝Έκε Ήζ┤ΈΠβΊΧ┤ΉΧ╝ΊΧσΈΜΙΈΜν.

ΉΨ┤ΈΨν Έ░σΈ▓ΧΉζΕ ΊΔζΊΧαΈΥι ΉΩΕΉ▓φ Έ╣ΕΉΜ╝ ΉηΣΉΩΖΉηΖΈΜΙΈΜν. Ά╖╕ΈουΆ│ι Έ╣ΕΉΜ╝ ΉηΣΉΩΖΉζΑ ΉπΑΉΨΣΊΧ┤ΉΧ╝ΊΧσΈΜΙΈΜν. ΈφΡ, Ά░ΑΈΒΦΉζΑ Ά╖╕ΈΔξ ΈΙΙΈυ╝ΉζΕ Έρ╕Ά╕ΙΆ│ι ΊΧ┤ΉΧ╝ΊΧαΉπΑΈπΝΉγΦέΑο

Ήζ┤ ΉΔΙΈκεΉγ┤ ΈσΦΈςρΈου Έ╕ΦΈκζΉζΑ Έ│┤ΊΗ╡ Ήζ┤ΉιΕ Έ╕ΦΈκζΉζα 2Έ░░ ΊΒυΆ╕░Έξ╝ Ά░ΑΉπΣΈΜΙΈΜν. Ά╖╕ΈιΘΆ╕░ΉΩΡ, Ήγ░ΈουΉζα vectorΆ░Α size()ΈΠΕ capacity()ΈΠΕ 100Ά░εΉζα elementΈξ╝ Ά░ΑΉπΑΆ│ι ΉηΙΉζΕ ΈΧΝ Ήγ░ΈουΆ░Α push_back()ΉζΕ ΊΧ┤Έ▓ΕΈουΈσ┤, ΉΔΙΈκεΉγ┤ ΈσΦΈςρΈου Έ╕ΦΈκζΉζΑ 200Ά░εΉζα elementΈξ╝ Ά░ΑΉπΙ ΉΙα ΉηΙΆ▓Ν ΊΧιΈΜ╣ΈΡσΈΜΙΈΜν.

200Ά░εΉζα ΈσΦΈςρΈου Έ╕ΦΈκζΉζΕ Ά░ΑΉπΑΈΛΦΆ▓Ν ΉηαΈς╗ΈΡυΈΜνΈΛΦΆ▓Ν ΉΧΕΈΜβΈΜΙΈΜν. ΈΜνΈπΝ Ήζ┤ allocation ΉηΣΉΩΖΉζ┤ Έ░αΈ│╡ΊΧ┤ΉΕε ΈΓαΊΔΑΈΓεΈΜνΈσ┤ ΉΩΕΉ▓φΈΓαΆ▓Ν ΈΛΡΈινΉπΑΆ▓ιΉμι. Ήζ┤Έθ░ allocation Ά│╝ΉιΧΉζΕ ΊΦ╝ΊΧαΆ╕░ ΉεΕΊΧ┤ Ήγ░ΈουΈΛΦ ΈσΦΈςρΈου Έ╕ΦΈκζΉζΕ έΑαΈψ╕Έου ΊΧιΈΜ╣έΑβ (i.e. reserve) ΊΧι ΉΙα ΉηΙΉΛ╡ΈΜΙΈΜν.

ΉΧΕΈηα Ή╜ΦΈΥεΈξ╝ ΊΧεΈ▓Ι Έ┤ΖΉΜεΈΜν.

Ή░ρΉζ┤Έξ╝ ΊΧεΈ▓Ι Έ│┤ΉΕ╕ΉγΦ! ElementΆ░Α 100Ά░ε Έ░ΨΉΩΡ ΉΧΙΈΡαΈΛΦΈΞ░ Ήζ┤ΈιΘΆ▓Ν Ή░ρΉζ┤Ά░Α ΈΓσΈΜΙΈΜν.

ElementΉζα ΉΙαΆ░Α ΈςΘΆ░εΆ░Α ΈΡαΈΔΡΉΩΡ ΈΦ░Έζ╝ΉΕε reserveΉζα ΉΕ▒ΈΛξ ΊΨξΉΔΒΉ╣αΆ░Α ΈΜυΈζ╝ΉπΙ ΉΙα ΉηΙΉΛ╡ΈΜΙΈΜν. ΊΧαΉπΑΈπΝ ΊΧαΈΓα ΊβΧΉΜνΊΧε ΉιΡΉζΑ, reserveΈξ╝ ΉΥ░Έσ┤ Έυ┤Ήκ░Ά▒┤ Έ╣ρΈζ╝ΉπΕΈΜνΈΛΦ Ά▓ΒΈΜΙΈΜν.

Ά╖╕ΈουΆ│ι ΈαΡ, ObsessiveReclycing ΊΧρΉΙαΈξ╝ ΉηΡΉΕ╕ΊηΙ Έ┤ΖΉΜεΈΜν. ΉηΣΉζΑ vectorΉΩΡΉΕεΈΛΦ ΉΕ▒ΈΛξΉ░ρΉζ┤Ά░Α ΉηΙΈΛΦΈΞ░, ΊΒ░ vectorΉΩΡΉΕεΈΛΦ Έ│Ε Ή░ρΉζ┤Ά░Α ΉΩΗΉΛ╡ΈΜΙΈΜν.

ΉΧΕ, Έυ╝Έκι ObsessiveRecyclingΉζ┤ ΈΞΦ Έ╣ιΈοΖΈΜΙΈΜν. Ά╖╕ΈΔξ ΉΨ┤ΈΨν ΉΓυΉζ┤ΉοΙΈξ╝ ΉΥ░ΈΔΡΉΩΡ ΈΦ░Έζ╝ΉΕε έΑαΈΙΙΉΩΡ ΈζΕΆ▓Ν ΈΜυΈζ╝ΉπΑΈΛΦΉπΑέΑβΈΛΦ Ήκ░Ά╕Ι Ή░ρΉζ┤Ά░Α ΈΓι ΉΙα ΉηΙΈΜνΈΛΦΆ▒░Ήμι.
