C++ Best Practices (0) - мӢңмһ‘н•ҳл©ҙм„ң...

Posted on 2021-09-25 Edited on 2022-04-22 In 2. Programming , 2.2 C++ , C++ Best Practices мӢңлҰ¬мҰҲ Views:

мӢңлҰ¬мҰҲлҘј мӢңмһ‘н•ҳл©ҙм„ң

м ң мЈјліҖм—җлҠ” кё°м—…/лҢҖн•ҷмӣҗм—җм„ң м—°кө¬лҘј н•ҳмӢңлҠ” л¶„л“Өмқҙ л§Һмқҙ кі„мӢӯлӢҲлӢӨ.

кіөл¶ҖлҸ„ л§Һмқҙ н•ҳмӢңл©ҙм„ң state-of-the-art м•Ңкі лҰ¬мҰҳмқ„ к°ңл°ңн•ҳмӢңлҠ” л¶„л“Өмқҙ л§ҺмңјмӢ лҚ°, көүмһҘнһҲ мқҳмҷёмқё м җмқ„ н•ҳлӮҳ л°ңкІ¬н–ҲмҠөлӢҲлӢӨ.

Software engineeringм—җ м•Ҫн•ҳлӢӨ.

мҷң к·ёлҹ°кұёк№Ңмҡ”?

к°ңмқём ҒмңјлЎң лҠҗлҒјкё°м—җлҠ” л§ҺмқҖ л¶„л“Өмқҙ лҢҖн•ҷкөҗ->лҢҖн•ҷмӣҗ->м—°кө¬мҶҢ н…ҢнҒ¬нҠёлҰ¬лҘј нғҖл©ҙм„ң, кё°м—…м—җм„ң мӨ‘мҡ”мӢңн•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙ м—”м§ҖлӢҲм–ҙл§Ғмқ„ кІӘм§Җ лӘ»н•ҳл©ҙм„ң мғқкё°лҠ” л¬ём ңмқё кІғ к°ҷмҠөлӢҲлӢӨ.

нҠ№нһҲлӮҳ м»ҙн“Ён„° кіөн•ҷмқ„ м „кіөн•ҳм§Җ м•Ҡм•„лҸ„ м§„н•ҷн•  мҲҳ мһҲм§Җл§Ң, C++ мҪ”л”©мқ„ л§Һмқҙ н•ҙм•јн•ҳлҠ” л¶„м•јм—җм„ң мқҙлҹ¬н•ң лӘЁмҠөмқҙ л§Һмқҙ ліҙмһ…лӢҲлӢӨ (e.g. лЎңліҙнӢұмҠӨ л¶„м•ј -> м „кё°м „мһҗкіөн•ҷ мЎём—…мғқ)

Software engineeringмқ„ кіөл¶Җн•ҙлҸ„ мўӢм§Җл§Ң, лҢҖл¶Җл¶„мқҳ мӮ¬лһҢл“ӨмқҖ мӢңк°„мқҙ м—ҶмҠөлӢҲлӢӨ.

 

к·ёлҹ¬лҜҖлЎң мқҙлІҲ мӢңлҰ¬мҰҲм—җм„ңлҠ” м•Ңм§ңл°°кё° лҠҗлӮҢмңјлЎң C++ м–ём–ҙлЎң н”„лЎңк·ёлһҳл°Қмқ„ н•  л•Ң к¶ҢмһҘлҗҳлҠ” кё°лІ•м—җ лҢҖн•ҙ мҶҢк°ңн•©лӢҲлӢӨ.

мқҙ кё°лІ•л“Өмқ„ мӮ¬мҡ©н–Ҳмқ„ л•Ң кё°лҢҖн•  мҲҳ мһҲлҠ” м җл“ӨмқҖвҖҰ

  • мҙҲліҙк°Җ н•ҳкё° мү¬мҡҙ мӢӨмҲҳл“Өмқ„ н”јн•  мҲҳ мһҲмҠөлӢҲлӢӨ
  • мҪ”л“ңм—җм„ң лІ„к·ёлҘј м°ҫкё° мү¬мӣҢм§‘лӢҲлӢӨ
  • мўӢмқҖ м„ұлҠҘмқ„ мң м§Җн•ҳл©ҙм„ң мҪ”л“ң нҖ„лҰ¬нӢ°лҘј лҶ’мқј мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙ мӢңлҰ¬мҰҲлҘј м Ғмқ„ л•Ң Jason Turnerмқҳ C++ Best Practices мұ…мқҳ лӘ©м°ЁлҘј л§Һмқҙ м°ёкі н–ҲмҠөлӢҲлӢӨ.

 

C++ мҪ”л”©мқҳ нҠ№м§•

м—¬лҹ¬л¶„мқҙ C++лЎң мҪ”л“ңлҘј м§ңлҠ” мқҙмң лҠ” лӢЁ н•ҳлӮҳ л•Ңл¬ёмқјкІҒлӢҲлӢӨ.

м„ұлҠҘ л•Ңл¬ёмқҙм§Җмҡ”.

C++мқҖ көүмһҘнһҲ low-levelн•ң мҲҳмӨҖк№Ңм§Җ мҪ”л“ңлҘј м§Ө мҲҳ мһҲкІҢ н•ҙмЈјм§Җл§Ң, к·ёл§ҢнҒј кі л Өн•ҙм•јн•  л¶Җл¶„мқҙ л§Һкі  мӢӨмҲҳн•ҳкё° мүҪмҠөлӢҲлӢӨ.

лӢӨлҘё м–ём–ҙліҙлӢӨ лҚ” мўӢмқҖ м„ұлҠҘмқ„ м–»кё° мң„н•ҙ C++мқ„ мҚјм§Җл§Ң, лҚ” мң„н—ҳн• мҲҳлҸ„ мһҲлӢӨлҠ” кІғмқҙм§Җмҡ” (н•ҳмқҙлҰ¬мҠӨнҒ¬, н•ҳмқҙлҰ¬н„ҙмқҙлқјкі лҸ„ ліј мҲҳ мһҲмҠөлӢҲлӢӨ).

мҡ°лҰ¬мқҳ C++ мҪ”л“ңм—җм„ң мӢӨмҲҳк°Җ мһҲмқ„ л•Ң, мӢӨм ң мӮ¬мҡ©мІҳм—җм„ңлҠ” мқҙлҹ¬н•ң кІҪмҡ°к°Җ лӮҳнғҖлӮ©лӢҲлӢӨ.

  • кІҢмһ„л¶„м•ј : лІ„к·ёк°Җ лӮҳнғҖлӮ©лӢҲлӢӨ. кІҢмһ„м„ұмқҙ л–Ём–ҙм§ҲмҲҳлҸ„ мһҲкі , кІҢмһ„мқҙ кәјм§ҲмҲҳлҸ„ мһҲкі , н•ҙнӮ№мқ„ лӢ№н• мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.
  • кёҲмңөл¶„м•ј : мһҳлӘ»лҗң кІ°м •мқ„ лӮҙл Ө лҸҲмқ„ мһғмҠөлӢҲлӢӨ. лӮҙ кІ°м •мңјлЎң мқён•ҙ мӢңмһҘмқҳ лӢӨлҘё кІ°м •л“ӨлҸ„ мҳҒн–Ҙмқ„ л°ӣмқ„ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.
  • лЎңліҙнӢұмҠӨ : лӢ№мӢ мқҳ лЎңлҙҮ/л“ңлЎ мқҙ мӮ¬кі лҘј кІӘм–ҙ лҚ” мӣҖм§Ғмқј мҲҳ м—ҶмҠөлӢҲлӢӨ.
  • мһҗмңЁмЈјн–ү/мҡ°мЈјн•ӯкіө : лӢ№мӢ мқҳ мһҗлҸҷм°Ё/лЎңмј“мқҙ мӮ¬кі лҘј кІӘм–ҙ нҸҗкё°н•ҙм•јн•©лӢҲлӢӨ. мқёлӘ…мӮ¬кі к°Җ мһҲмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙлҹ¬н•ң C++мқҳ нҠ№м„ұмқ„ кі л Өн–Ҳмқ„ л•Ң, мҪ”л“ң нҖ„лҰ¬нӢ°лҘј лҶ’мқҙкі  мўӢмқҖ мҪ”л“ңлҘј м Ғм–ҙм•јн•ҳлҠ”кұҙ мӨ‘мҡ”н•©лӢҲлӢӨ.
