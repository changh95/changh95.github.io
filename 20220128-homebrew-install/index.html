Rosetta 2 설치

Rosetta가 설치되고나면, x86용 Terminal 앱을 만들자.

우선 Terminal 앱을 검색해서 찾고, 클론을 만든다음 이름을 ‘Terminal (x86)’으로 바꾼다.

이 클론에 마우스 오른쪽 클릭 -> Get Info를 선택하고 Open using Rosetta를 한다.
